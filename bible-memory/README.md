# 말씀 암송 (Bible Memory)

주제별 성경 암송 웹앱. 서버·로그인 없이 정적 파일만으로 동작하며, GitHub Pages에 그대로 올려 링크로 공유할 수 있습니다.

본문은 **개역한글(1961)** 기준이며, 여러 번역본을 나중에 추가할 수 있는 구조로 설계돼 있습니다.

## 기능
- **둘러보기** — 컬렉션(5·8·60·100·DEP242·180·900구절) → 분류 계층 트리 → 구절 카드
- **암송** — 플래시카드 + 힌트 4단계(전체 / 첫글자 / 빈칸 / 참조만), 셔플, "외웠어요" 체크
- **테스트** — ①참조→구절(자가채점) ②구절→참조(4지선다) ③빈칸 채우기, 점수·기록 저장
- **진도** — 컬렉션별 진도율, 진도는 각자 브라우저(localStorage)에 저장
- **번역본 선택** — 상단 컬렉션 버튼 → "번역본"에서 전환(현재 개역한글, 추후 추가 가능)
- **PWA** — 폰에서 "홈 화면에 추가" 시 앱처럼 사용, 오프라인 동작

## 파일 구성
| 파일 | 설명 |
|------|------|
| `index.html` | 앱 전체 (거의 수정할 일 없음) |
| `data.js` | **앱이 읽는 구절 데이터** (compact) |
| `data/verses.json` | 같은 데이터의 보관용 pretty JSON (사람이 읽기 좋음) |
| `manifest.webmanifest`, `sw.js`, `icon.svg` | PWA용 |

> `data.js`와 `data/verses.json`은 같은 내용입니다. 수정 시 둘을 맞춰 주세요(또는 `data.js`만 쓰고 json은 백업으로 둬도 됩니다).

## 데이터 구조
`data.js`는 `window.BIBLE_DATA = { ... }` 형태입니다.

```js
{
  "meta": { "translation": "개역한글", "total": 1495, ... },
  "versions": [ { "id": "krv", "name": "개역한글", "year": 1961 } ],
  "defaultVersion": "krv",
  "collections": [ { "name": "확립 100구절", "short": "100구절 · 확립", "count": 100 }, ... ],
  "verses": [
    {
      "id": 231, "coll": "확립 100구절",
      "sub": ["구원의 확신", "은혜로 얻은 구원"],
      "book": "에베소서", "chap": 2, "verse": "8-9",
      "ref": "에베소서 2:8-9",
      "texts": { "krv": "너희가 그 은혜를 인하여 ..." }
    }
  ]
}
```
- `coll` : 컬렉션 이름 (`collections`의 `name`과 일치)
- `sub` : 분류 계층 배열 (상위→하위). 트리·암송/시험 범위가 이걸로 묶입니다.
- `ref` : 화면 표시용 장절. `book/chap/verse`는 시험 기능용 분해값.
- `texts` : **번역본별 본문**. 키가 `versions[].id`와 대응됩니다.

### 번역본 추가하는 법 (예: 개역개정)
1. `versions`에 `{ "id": "krv2", "name": "개역개정", "year": 1998 }` 추가
2. 각 구절 `texts`에 `"krv2": "..."` 채우기
3. 끝. 앱 상단에서 번역본을 고르면 해당 본문으로 표시됩니다.

## 데이터 출처 & 검증
- 원본: 네비게이토 주제별 성경 암송 카드집(5·8·60·100·180·900구절, HWP에서 추출) + DEP 242구절(성경별 암송 목록). 총 **1,495구절**.
- 검증: 개역한글 전문과 장·절 대조 → 98.6% 정확 일치로 개역한글임을 확인. 카드 오탈자 2건(`잃을 것잉→것이요`, 책이름 `잠엄→잠언`) 교정.

## GitHub Pages 배포
1. 이 저장소 **Settings → Pages → Branch: `main` / 폴더 `/root`** 선택 후 저장
2. 잠시 뒤 `https://ella16.github.io/bible-memory/` 로 접속 → 링크 공유
   - (private 저장소는 Pages가 유료 플랜에서만 공개됩니다. 무료로 공개하려면 저장소를 public으로 바꾸거나, 배포용만 따로 두세요.)

## 로컬에서 열기
`index.html`을 브라우저로 열면 바로 동작합니다(데이터는 `<script>`로 불러와 서버 불필요).
PWA/서비스워커는 `https` 또는 `localhost`에서만 켜집니다.
