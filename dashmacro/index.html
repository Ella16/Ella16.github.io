DashMacro

유저스크립트를 등록해 원하는 사이트에서 자동 실행하는 크롬 확장 프로그램

기능

유저스크립트 관리 — 표준 ==UserScript== 메타데이터를 그대로 사용합니다.
사이트별 자동 실행 — @match 패턴으로 실행할 사이트를 지정하세요.
내장 편집기 — 확장 프로그램 안에서 바로 스크립트를 작성하고 수정합니다.
완전 로컬 — 모든 스크립트는 브라우저 안에만 저장되고, 어떤 데이터도 외부로 전송하지 않습니다.
