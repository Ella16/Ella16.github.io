DashMacro 스크립트 작성 가이드

DashMacro에서 동작하는 유저스크립트를 직접, 또는 AI 도구로 만들 때 참고하세요.

🤖 Claude Code / AI로 만드시나요? 이 스펙 파일(llms.txt)을 통째로 붙여넣으면 DashMacro에 맞는 스크립트를 정확히 생성합니다.

메타데이터

스크립트 맨 위에 메타 블록을 넣습니다. DashMacro가 지원하는 키만 동작합니다.

// ==UserScript==
// @name         내 스크립트
// @version      1.0
// @match        https://example.com/*
// @exclude      https://example.com/logout*
// @run-at       document-idle
// ==/UserScript==
키설명
@nameDashMacro 목록에 표시되는 이름
@version선택. URL로 설치한 경우 업데이트 확인에 사용
@match / @includeChrome 매치 패턴(정규식 아님). 여러 개 가능. 없으면 모든 사이트에서 실행
@exclude / @exclude-match제외할 매치 패턴
@run-atdocument-start / document-end / document-idle(기본)

@grant, @require, @resource, @namespace 등은 무시됩니다(넣어도 무해).

실행 모델 (꼭 알아둘 것)

  • 격리된 월드에서 실행됩니다. DOM은 읽고 바꿀 수 있지만, 페이지 자체의 JS 전역변수는 볼 수 없습니다. unsafeWindow도 격리 월드의 window라 페이지 전역엔 접근 못 합니다.
  • 외부 라이브러리 불가. @require 미지원 — jQuery 등은 인라인으로 넣어야 합니다.
  • 스크립트는 전체 페이지 로드당 한 번 주입됩니다. SPA(React/Vue 등)의 화면 전환에는 다시 실행되지 않으므로 직접 감지해야 합니다.

GM_ 함수

소스에 GM_ 또는 unsafeWindow가 있으면 자동 주입됩니다.

  • GM_getValue / GM_setValue / GM_deleteValue / GM_listValues — localStorage 기반. 사이트별로 격리됩니다(다른 유저스크립트 매니저의 크로스사이트 저장과 다름).
  • GM_addStyle(css), GM_log(...)
GM_xmlhttpRequest는 미지원입니다(크로스오리진 차단). 같은 출처면 fetch를 쓰세요.

자주 필요한 패턴

1. 요소가 나타날 때까지 대기

async function waitFor(sel, timeout = 8000) {
  const start = Date.now();
  while (Date.now() - start < timeout) {
    const el = document.querySelector(sel);
    if (el) return el;
    await new Promise(r => setTimeout(r, 200));
  }
  return null;
}

2. SPA 화면 전환 감지

function onRouteChange(fn) {
  const wrap = (t) => { const o = history[t];
    history[t] = function () { const r = o.apply(this, arguments);
      window.dispatchEvent(new Event('dm:nav')); return r; }; };
  wrap('pushState'); wrap('replaceState');
  addEventListener('popstate', () => dispatchEvent(new Event('dm:nav')));
  addEventListener('dm:nav', () => setTimeout(fn, 500));
}

3. React 입력창에 값 넣기

function setNativeValue(input, value) {
  const setter = Object.getOwnPropertyDescriptor(
    window.HTMLInputElement.prototype, 'value').set;
  setter.call(input, value);
  input.dispatchEvent(new Event('input',  { bubbles: true }));
  input.dispatchEvent(new Event('change', { bubbles: true }));
}
