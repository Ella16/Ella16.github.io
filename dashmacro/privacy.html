DashMacro Privacy Policy

Effective date: July 21, 2026

Information We Collect

DashMacro does not collect any personal information. It has no analytics, no tracking, and no user accounts.

Data Storage

The user scripts you write and your settings are stored solely in your browser's local storage (chrome.storage.local). They are never transmitted to any external server, including the developer's.

Network Requests

The extension retrieves a small static file from the developer's site (ella16.github.io) to update the text of the in-extension notice banner. This request contains no information that can identify you; it is a plain download of a public file.

Permissions

  • userScripts — used to register and run the scripts you add, on the sites you specify.
  • storage — used to save your scripts and settings locally.
  • Host permissions — required to run your scripts on the sites you designate with @match. The extension does not modify any page other than through the scripts you have explicitly enabled.

User-Provided Scripts

DashMacro is a tool that runs scripts you choose to install. Scripts you write or import are executed as-is. You are responsible for the behavior of any script you add, including scripts obtained from third parties. DashMacro itself does not send your data anywhere.

Changes to This Policy

If future versions add features that involve additional data handling, this policy will be updated and the effective date above will be revised.
